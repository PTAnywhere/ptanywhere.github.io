PTAnywhere development

Documentation for PTAnywhere developers

This website includes very specific and technical information about PTAnywhere. If you are a user or somebody trying to understand what is PTAnywhere, please visit its official website.

Here we will explain what are the Github projects in PTAnywhere organization, how to use them and how to contribute to them.

PTAnywhere is composed by...

  • Scripts which ease the installation and deployment of all its components
  • An HTTP API for Packet Tracer
  • An internal HTTP API for creating different instances of Packet Tracer
  • A set of widgets to interact with Packet Tracer through the web

Additionally, we have also made public the following related projects:

  • performance-testing is a Python project created to measure the performance of PTAnywhere (specifically the Docker-based Packet Tracer container).
  • usage-analysis is an outdated web application which visualises users interaction with the widgets.
